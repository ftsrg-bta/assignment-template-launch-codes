## Repository structure

* Chaincode implementation: `chaincode/` directory
* Test network: [`fablo-config.json`](fablo-config.json) (managed by [Fablo](https://github.com/hyperledger-labs/fablo))
* Test call collection: `test-calls/` directory
* Network interaction script: [`scripts/cc.sh`](scripts/cc.sh)
